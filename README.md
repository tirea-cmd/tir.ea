## 🚀 Upload pe GitHub Pages

1. Șterge tot ce ai în repo-ul vechi
2. Urcă toate cele 17 fișiere
3. Commit
4. Așteaptă ~1 min

## 📲 Pe telefon (dacă ai deja PWA instalat)

1. Dezinstalează PWA-ul vechi (ține apăsat pe iconiță → Uninstall)
2. Chrome → Settings → Privacy → Clear browsing data pentru site-ul tău
3. Deschide linkul → Install app
